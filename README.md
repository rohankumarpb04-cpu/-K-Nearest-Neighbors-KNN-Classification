# 🌸 Iris Flower Classification using K-Nearest Neighbors (KNN)

## 📌 Project Overview
This project implements the K-Nearest Neighbors (KNN) algorithm to classify Iris flowers into three species based on their measurements. The project includes data preprocessing, feature scaling, model training, evaluation, and visualization.

---

## 🎯 Objective
- Load and preprocess the Iris dataset.
- Normalize features using StandardScaler.
- Train a KNN classifier.
- Experiment with different K values.
- Evaluate the model using accuracy and confusion matrix.
- Visualize model performance.

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset
**Dataset:** Iris Dataset

The dataset contains four numerical features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

---

## 🚀 Workflow
1. Import libraries.
2. Load the dataset.
3. Split features and target.
4. Standardize the data.
5. Train the KNN classifier.
6. Predict species.
7. Evaluate using accuracy, confusion matrix, and classification report.
8. Compare different K values.
9. Visualize results.

---

## 📊 Evaluation Metrics
